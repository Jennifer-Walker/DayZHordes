# SJHorde

This directory contains the file structure for the DayZ mod **SJHorde**.

- `Addons` – compiled PBO files.
- `Keys` – server keys for verifying the mod.
- `Scripts` – source scripts divided into `3_Game`, `4_World`, and `5_Mission`.
- `mod.cpp` – basic mod configuration.
- `meta.cpp` – mod metadata.

Place your actual mod files within these folders as needed.
